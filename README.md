# AI Career Path Predictor

An AI-powered web application that predicts suitable career paths based on a user's skills or uploaded resume.

## Live Application
https://ai-career-path-predictor-b7a5kgppxvmh6ripqetdpa.streamlit.app/

## Features

- Enter skills manually
- Upload resume (PDF)
- Automatic skill detection
- Career path prediction
- Skill gap analysis
- Career roadmap generation
- Career match visualization

## Tech Stack

Python  
Streamlit  
PyPDF2  
Matplotlib  

## How it Works

1. User enters skills or uploads a resume
2. The system extracts detected skills
3. Skills are compared with predefined career requirements
4. Career match scores are calculated
5. The system suggests suitable career paths and a learning roadmap

## Example Careers Supported

- Data Scientist
- Machine Learning Engineer
- Web Developer
- Data Analyst
- AI Engineer
